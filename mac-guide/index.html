Mac 入门指南

发表于 2019-10-09 分类于 Skill

一、你好，Mac

1、Mac 是什么？

Mac 就是苹果电脑，确切的说是苹果公司生产的运行 macOS（苹果电脑系统）的电脑。现在官方在售的 Mac 包括笔记本、一体机和台式机。

Mac 笔记本有 MacBook、MacBook Air、MacBook Pro。因为便携，大多数个人都会选择笔记本。

Mac 一体机叫 iMac。外表简洁，看起来就像是一个显示器，但实际已经包含 Mac 主机、音响、摄像头、网卡等完整电脑的功能。大致包括 21 寸 的 4K 屏幕款和 27 寸的 5K 款。针对专业级用户，还有性能更强大的 iMac Pro。

Mac 主机有 Mac mini 和 Mac Pro。注意 MacBook Pro 和 Mac Pro 是两个产品。

2、Mac 运行的系统叫 macOS

类似 PC 电脑的 Windows 10， 苹果电脑运行的系统叫 macOS。macOS 是 Mac 的灵魂，也是 Mac 用户每天要用到的部分。macOS 和 Windows 是两个截然不同的系统，使用 macOS 时，你需要抛弃使用 Windows 的固有思维习惯。像第一次接触电脑时那样开始用 macOS，你才能体会到 Mac 的专注、高效和特别之处。

进入 macOS，点击桌面最左上角的  图标，选择「关于本机」，可以看到你的 Mac 用的是哪一款、用的什么处理器、内存和硬盘有多大容量、系统版本等等。就像是你的 Mac 的身份证。

3、macOS 和 Windows 有什么不同？

软件不通用，Windows 软件的扩展名后缀通常是 .exe ，macOS 是 .app。很多软件会同时提供 Windows 和 Mac 版本，在 macOS 上下载 Mac 版本才能安装。

文件系统不一样，Mac 采用 APFS 格式，Windows 用的是 NTFS 格式。苹果格式的磁盘默认在 Windows 上不能读写，Windows NTFS 格式磁盘在 Mac 上也只能读不能写。但是可以有方法实现双平台支持，后面再说。对了，macOS 一般不需要磁盘分区。

键盘和快捷键不完全一样，但是大体还是相似的，这个也后面再细说。

macOS 是免费的，Windows 通常是付费的。

Mac 上支持安装 Windows 10 系统，但是无法充分发挥 Mac 的优势，一般不推荐这样用。PC 机上安装 macOS 俗称「黑苹果」，并没有官方支持，一般也不推荐。如果习惯用 Windows，就好好挑一台 PC 吧。如果想用好 Mac，使用自家的 macOS 自然是最好的选择。

两个系统不同还有很多，比如文件管理、窗口管理。但是差别也没那么大。只是在决定拥有一台 Mac 前，先搞清楚是不是适合自己，是不是能满足自己的使用需求，才是最应该关心的事。

如果你有一台 iPhone，用使用 iOS 的思维去理解 macOS，也会更方便一些。

4、Mac 适合哪些人？

先来说说 Mac 不适合哪些人。

如果你是游戏党，那 Mac 很大可能并不是最好的选择，很多游戏没有原生 Mac 版本。但也并不是说 Mac 上无游戏可玩，像荒野行动、Dota 2、刺客信条 2，暴雪家的炉石、魔兽等都有 Mac 版，Steam 上也有很多 Mac 游戏。但我仍然认为 Mac 不是一台合格的「游戏机」。普通 Mac 的显卡配置和散热架构就不适合玩游戏，并且很多游戏只有在高端显卡 PC 机才能享受更高的分辨率与帧数支持。苹果一直强调的是把 Mac 用作专业图片处理、视频剪辑、VR 内容创作等生产力场景，而不是一个游戏机。

如果你的学科、工作需要一些特定软件，但是这些软件只有 Windows 版本，那买一台 Windows PC 会更适合。虽然 Mac 也可以安装 Windows，但是并不推荐把 Windows 作为 Mac 的主力系统。

如果你预算不高（5000 以下？），PC 机会有更多便宜的选择。

除去以上几种情况，我认为 Mac 几乎适合所有人。

不管是日常上网、办公，还是自媒体写作、摄影后期、视频剪辑、软件开发、会议演示等，Mac 都完全可以胜任。这时选择 Mac 还是 windows PC，只取决于你喜欢哪个。如果恰好你在用 iPhone，那电脑选择 Mac 就能体会到苹果产品生态的强大和魅力之处。

二、拿到 Mac 以后

1、登录

推荐用你的 Apple ID 登录 Mac，不仅可以备份和同步文件，还能体验到苹果软件生态的统一性和高效。然后记得给 Mac 设置一个 6 位数的登录密码，一定要不容易被猜到，同时自己不会忘记。如果你用的是带触控 ID 的 Mac 笔记本，可以设置指纹密码来解锁 Mac。

2、认识桌面

桌面是软件的使用区域。用来放置其他软件窗口、文件等。macOS 支持多桌面，只要在触控板四只向上拖，就可以查看所有打开的桌面。四指左右拖可以切换桌面。也可以用快捷键 control - 方向键实现。

你可以在不同的桌面放不同的软件，这样便于管理软件窗口，也可以让使用更专注。

桌面壁纸可以换，可以用自己的图片作为壁纸。系统自带的图片壁纸非常漂亮，最新版 macOS 还支持动态壁纸，这不是普通的壁纸自动更换，而是会根据一天的时间产生细微的变化，值得试一试。

3、顶部菜单栏

macOS 桌面上有一条固定在最顶部的横着的菜单栏，它的用处非常大。

最左边的  图标，在上面提到过，点击可以查看 Mac 信息、打开「系统偏好设置」、睡眠、关机等操作。

紧接着是应用菜单，这一块是随着当前正在使用的软件而变化的。默认没有打开软件就会显示「访达」菜单。你可以试试点开不同的软件，这一块的菜单会发生不同的变化。

右边是状态菜单，放置一些很实用的快捷显示和操作。像连 Wi-Fi、调节音量、查看日期和时间、输入法状态等。

再往右就是「聚焦」搜索和 Siri，以及「通知中心」。

「通知中心」跟 iPhone 上的通知中心很像，通过点击最右侧图标或或者用双指从触控板右边缘向左轻扫打开。可以显示今天的日期、天气、日历事项等，还可以快捷开启「夜览」和「勿扰模式」。

4、程序坞

在桌面最下面一条放了很多软件图标的地方就是「程序坞」，不要问为什么叫这么奇怪的名字，就把它看作 iPhone 界面最下方的固定 App 快捷图标就行了。点击程序坞上的 App 图标就能打开这个 App。其中有个「启动台」可以显示所有安装的 App。最右边有「下载」文件夹和废纸篓。直接把文件或 App 拖到废纸篓就可以删除或卸载。

所以 macOS 主界面从上到下就是菜单栏、桌面以及程序坞三个部分。

5、启动台

点击程序坞的「启动台」图标，可以显示所有安装的 App。如果有多页，还可以左右滚动切换。有没有发现启动台是不是跟 iPhone 的界面很像！

6、访达

「访达」有点像 Windows 上的「文件资源管理器」，可以管理 Mac 上的一切内容。

7、系统偏好设置

「系统偏好设置」就像 Mac 系统的管家。在这里可以进行很多有用的设置。

可以设置桌面与屏保、自定义程序坞、设置语言和输入法、开启 iCloud 同步、设置打印机、更新系统等。建议每一个设置项都点开看一看，这样下次想修改某个设置时就知道在哪里了。

三、Mac 上的软件

Mac 上的软件跟 Windows 不通用，安装、卸载方式也不一样。Mac 软件通常给人的印象是贵、小而美、没有复杂的功能和广告。现在常用的软件基本都有 Mac 版。

需要说明一下，本文中提到的「软件」、「应用」、「程序」、「App」都是一个意思。

1、Mac 软件在哪下？

Mac App Store

就像 iPhone 安装软件有自己的 App Store（应用市场），Mac 也有自己的 App Store。

用你的 Apple ID 登录 App Store，就可以购买和下载上面的软件。有些软件是付费的，也有很多是免费的。用过 iPhone 的 App Store 很容易就理解了。

像微信、QQ、优酷、爱奇艺、WPS 等国内热门软件都可以在 App Store 搜索免费下载。

软件官网

由于 macOS 的软件沙盒机制和对权限接口的限制，很多软件因为一些功能无法上架 App Store，就提供官网下载。可以自行下载后安装。

建议优先从 App Store 搜索安装需要的 App，其次去软件官网下载。这里要留意有时候某搜索引擎会搜出假官网，这时候就要注意了。有条件的推荐使用更好的搜索引擎搜官网。

同时不建议从提供所谓破解、激活版下载软件，可能会同时安装到恶意软件，并且支持正版不仅是对软件开发者的尊重，也是对自己的时间和使用安全负责。

2、Mac 软件怎么安装？

macOS 里有一个「应用程序」文件夹，里面放的就是每个 .app 格式的 Mac 软件。所谓安装 Mac 软件，就是把软件 .app 文件放到「应用程序」文件夹中。

如果是从 App Store，点击获取或购买后就会自动安装到「应用程序」文件夹，这是最方便的方式。

如果是从网站下载的，情况就多一些。

很多时候我们从网站下载的软件包并不是 .app 格式，而是 .dmg 格式。这个是 Mac 的磁盘映像格式，你可以理解为是一个包，里面有真正用到的 .app 文件和其他一些东西（通常是应用程序文件夹的快捷方式）。

比如我要安装「虾米音乐」这个 App，从虾米官网下载到的是 XiamiMac-03051058.dmg 这个磁盘映像文件。

双击打开，就会在桌面显示磁盘，并打开安装窗口。一般都会有「应用程序」文件夹的快捷方式（图中的 Applications）。只要把「虾米音乐.app」拖到文件夹快捷方式中就可以了。

在桌面生成的磁盘可以拖拽到废纸篓或用 ⌘-E 卸载。

也有少数软件下载下来是一个 .pkg 的安装器软件包。双击会打开会显示安装窗口。只要一路点击继续安装，遇到需要输入密码时就输入 Mac 登录密码就可以了。

非 App Store 安装的软件，安装后第一次打开通常会问一句，点「打开」就好了。

3、打开 Mac 软件的 N 种方法

还以「虾米音乐」App 为例。

在「程序坞」点击软件图标可以打开。如果程序坞中不存在，可以用后面两个方法打开，然后右键点击图标，选择「在程序坞中保留」，这样以后就可以直接在「程序坞」上打开了。

使用「聚焦搜索」。这也是我常用的方式。点击桌面顶部菜单栏右侧的 🔍 图标，在弹出搜索框打上「虾米音乐」（甚至只输入「xia」都行），然后回车就可以打开。打开聚焦搜索我更推荐用快捷键（⌘-空格）。

从「启动台」打开。点击启动台，找到软件，点击打开。如果软件比较多，往往要找一下才能找到。

4、关闭软件

关闭软件有什么好讲的？但其实 Mac 新手很容易搞不懂「关闭」和「退出」的不同，并且不知道什么时候要关闭，什么时候要退出。

Mac 软件标准窗口左上角会有三个按钮，最左边红色代表「关闭软件窗口」，会关闭软件的操作，软件会保留在程序坞并且不会完全退出，再次点击程序坞上的软件图标能立即打开。有点类似于 iPhone 的 Home 键回到主屏幕，App 并没有退出。

如果想完全退出软件，可以在程序坞右键选择「退出」。

一般不用的软件，只需要关闭就好了，这样下次可以瞬间打开。Mac 有自己的内存管理优化机制，即使开启很多软件也不会很卡。但是如果软件不是常用的，或者占用内存比较大（如 Photoshop），在不用时就建议完全退出。

关闭软件窗口的快捷键是 ⌘-W，完全退出软件的快捷键是 ⌘-Q。

5、软件升级

在 App Store 安装的软件，可以直接在 App Store 更新。

自己下载安装的软件，一般在打开软件时，如果有新版本，会弹出更新窗口，然后根据提示更新就行了。或者在软件偏好设置看看能不能找到「检查更新」之类的选项。

6、软件偏好设置

一般 Mac 软件都有自己的「偏好设置」，可以在应用菜单栏打开设置（或 ⌘-逗号打开）。每个软件的设置项都不一样。

7、Mac 系统自带软件

macOS 内置了很多软件，有些非常好用。

「Safari」 是 Mac 默认浏览器。「QuickTime Player」 是视频播放器。「预览」可以查看和编辑 PDF 和图片文件。日历、照片、通讯录、邮件、信息等 App 可以通过 iCloud 账号跟其他 iCloud 设备（比如 iPhone）同步。

8、有哪些值得推荐的 Mac 软件？

说写比较常用的。

社交类的微信、QQ 就不多说了，可以在 App Store 免费下载。阿里旺旺只能在官网下载 Mac 版。

视频客户端有优酷、腾讯视频、爱奇艺，可以在 App Store 下载到。芒果 TV 要在官网下载。视频播放器推荐免费开源的 IINA，以及收费的 Movist，最近 QQ 影音播放器也不错。

网盘和下载工具。迅雷不用多说，掏钱才能体会到速度。百度网盘也有，腾讯微云也有，不过微云 App 做得不是很好。

还有一些常用的实用工具，像 The Unarchiver，免费解压缩软件，良心无烦人广告👍，几乎感觉不到它的存在。Magnet 是个分屏工具，大屏幕上用 macOS 可能更高效。这两个都可以在 App Store 下载。

办公软件，苹果自家的 iWork 值得一试，微软 Office 也有，WPS 也有 Mac 版。都可以在 App Store 下载。

知名的 Adobe 家族软件像 Photoshop、Lightroom、Premiere 都有。

四、更高效的使用 Mac

了解以上那些，基本上就能正常使用 Mac 来做一些事情。但是了解下面这些，能让你把 Mac 用的更高效、更爽。

1、快捷键

苹果键盘布局跟 Windows 键盘略有不同，快捷键也不太一样。

在苹果键盘上，通常顶行中会有一排特殊功能键，有调节音量的、调节显示屏亮度的、调节键盘灯的和其他功能图标。按图标键可执行相应功能，或将其与 Fn 键组合使用来用作 F1、F2、F3 或其他标准功能键。

左下角并排的三个修饰键，分别是 control ⌃、option ⌥、command ⌘。这三个键很重要。配合其他键能实现一些快捷操作个特殊功能。

这是另一个神奇的 Mac 独有的按键，通常需要配合其他键完成操作。记住多数快捷组合键都需要用到它就行了。

复制粘贴

⌘ + C 复制（拷贝） ⌘ + X 剪切 ⌘ + V 粘贴 ⌘ + option + V 粘贴，原复制对象不会保留，相当于「移动」

关闭和退出程序窗口

⌘ + W 关闭当前的软件窗口（软件并没有真正退出进程），相当于点了左上角的红色叉叉。 ⌘ + Q 真正退出软件。 ⌘ + option + esc 强制退出某个软件。通常在软件无响应时使用。

撤销

⌘ + Z 键撤销。在 Mac 下，撤销快捷键不仅可以撤销文本输入，甚至可以撤销刚刚不小心的操作！

比如你刚刚删除了一张图片，但是你马上又后悔了，你当然可以点开废纸篓，找到刚刚删除的图片，然后放回原处。或者在删除文件后马上点击 ⌘+Z，这时刚删除的文件又回到原来的位置！

全选

⌘ + A 全选，相当于 Windows 的 ctrl+A。

删除

⌘ + delete 删除，即把文件移至废纸篓。

保存

⌘ + S 保存。

新建

⌘ + N 新建，比如你在用 Safari，那 ⌘+N 就是新建一个 Safari 窗口。你在用 Photoshop，⌘+N 就是新建一个 PS 文稿。

快速打开聚焦搜索

⌘ + 空格键可以快速打开聚焦搜索框。聚焦搜索有时候非常高效。

快速切换软件

⌘ + tab 可以快速切换已经打开的软件，试一下你就明白了。

有些快捷键可以到系统偏好设置里修改成你喜欢（习惯）的，比如切换输入法，打开聚焦搜索等。

空格（space）键

这又是一个神奇的按键。在 macOS 里，几乎所有文件都可以直接用空格键快速查看预览。如视频、图片、音乐、页面、文档、PDF 等等。拿图片举例，只要点选文件夹里某张图片，点击空格键就打开了，再点击上下左右键可切换浏览其他图片。而不用每次都双击打开，再关闭，再双击打开另一张，再关闭……

绳命诚可贵，空格大法好。

2、触控板

在 MacBook 上能看到一个大大的触控板，可别小瞧了这个看似简单的区域，用好它能让你使用 Mac 的效率事半功倍，甚至可以完全取代鼠标，因为它更高效、更好用。用了这个东西才能真正体会到 Mac 系统的美妙，体会到什么是行云流水般自然顺畅。轻按、滚动、开合和轻扫，一指、两指、三指、四指。花 10 分钟了解这些手势，真的可以让你跟其他用 Mac 的新手不一样。

在「系统偏好设置」-「触控板」里，有详细的触控板设置和手势演示动画。强烈建议每一个手势都试一下，然后多用就自然记住了。

Force Touch 触控板有个黑科技就是它并不是机械结构的按下，而是通过 检测你按下去的力度进行感知，并通过线性马达带来震动反馈，让你真的以为是「按下去」的。这个在 Apple Watch 和 部分 iPhone 上也采用过。

Mac 触控板原本已经甩 PC 好几条街，新的 Force Touch 恐怕会让 PC 触控板更加无地自容。

你可以试试把 Mac 关机，在关机状态下点按触控板，你会惊叹的。

3、聚焦搜索

「聚焦搜索」真的是 macOS 上太好用的功能了！不仅能快速打开软件、当计算器、查找文件，还能查汇率，算单位。遇到像「1公斤等于多少磅？」或者「1美元等于多少人民币」这样问题还用得着百度吗？

可以点击顶部菜单栏右侧的放大镜图标打开，最快的也是我推荐的打开方式是用快捷键 ⌘-空格。

4、快速查看

「快速查看」（Quick Look）是苹果公司开发的一种文件预览技术。在 macOS 中，它能做到不使用软件打开就能直接用空格键预览文件，比如照片、声音、PDF、HTML、部分格式的视频、文档等。可以说是 macOS 上非常方便的一个特性，它有个更传神的名字叫「一指禅」。

拿图片举例，只要点选文件夹里某张图片，点击空格键就打开了，再点击上下左右键可切换浏览其他图片。而不用每次都双击打开，再关闭，再双击打开另一张，再关闭……

还可以通过安装插件来增强「快速查看」的功能。不仅能支持更多的文件格式预览，还能显示预览文件更全面的信息，使快速查看功能更加实用方便。你还可以看看这篇文章。

5、隔空投送

用「隔空投送」（AirDrop） 可以方便地在苹果设备之间互传文件，如 Mac、iPhone、iPad 之间无线分享照片、视频、网站、位置等内容。速度还很快，文件不会被压缩。

使用「隔空投送」时需要开启 Wi-Fi 开关和蓝牙。不过 Macbook 和 iPhone 的蓝牙都是低功耗的，一直开着也不会很费电。

「隔空投送」也不消耗流量，我试过在野外没网的情况下，两台 iPhone 或 Mac 之间也可以用 AirDrop 文件互传。

在 Mac 上使用「隔空投送」传文件很简单，可以在「访达」里选择文件，然后点击工具栏的分享图标，选择「隔空投送」就可以了。或者选择文件点右键，选择「共享」-「隔空投送」。

6、Handoff 连续互通

「连接互通」并不是一个软件，而是 macOS、iOS 和 watchOS 上的一系列互通功能。可以让不同的苹果设备无缝衔接使用，有时能大大提高使用效率。善用「隔空投送」和「连接互通」才能体会到苹果生态的好处。

使用这一功能，首先要你的苹果设备都登录同一个 Apple 账号，并且开启了「接力」。macOS 在「系统偏好设置」-「通用」中开启。

比如你正在 Mac 的 Safari 打开一个网页，在 iPhone 上可以快速打开这个网页。在 iPhone 上写了一半的备忘录笔记，可以在 Mac 上继续。iPhone 来电话了，可以直接在 Mac 上接听。在 iPhone 上复制一段文字或一张照片，可以在 Mac 上直接粘贴过来。还比如如果你带了 Apple Watch，打开 Mac 时会自动帮你解锁 Mac。这样的操作还很多，慢慢你就会发现发现。

7、截屏和录屏

Mac 上有个叫「屏幕快照」的系统软件，可以截屏和录屏。可以用聚焦搜索打开「屏幕快照」，但我更推荐用快捷键 Shift-Command-5 打开。

从左到右可以选择截全屏、截软件窗口、截手动选择区域、录制全屏、录制选择区域。「选项」里还可以设置录屏开始倒计时和保存位置。

另外还可以用快捷键 Shift-Command-3 快速截全屏，用快捷键 Shift-Command-4 截选择区域，用 Shift-Command-4 然后再按空格键选择窗口截屏。如果你是带触控栏的 MacBook Pro，还可以用快捷键Shift-Command-6给触控栏截屏。

五、你一定关心的一些问题

好了，到现在为止，你已经基本会用 Mac 了来做很多事情了。下面再聊几个你可能比较关心的问题。

1、Mac 需要关机吗？

我的建议是：日常使用不需要关机。在不用时 Mac 笔记本盒上盖子就好了，Mac 台式机可以设置睡眠。

如果有系统更新或者电脑故障需要关机重启，或者你可能很多天都不会用它，可以关机。

有些人关机只是出于习惯，并不知道为什么要（或不要）关机，只是关机让自己感觉上更好。想象我们怎么用手机的，不用时也不会关机，只需要按电源熄屏幕就好了。Mac 也是类似。

有些人会觉得不关机会让系统变慢。实际上 macOS 有很完善的系统维护机制，日常使用中我们感知不到它们的存在，却让我们的 Mac 始终处于一个良好的状态。

还有些人觉得不关机会费电。实际上 Mac 睡眠模式下消耗的电量非常少，如果想节能省电，还不如把空调温度调高 1 度来的实际。

每次关机和开机都要等待，每天都要在这上面浪费几分钟。某些电脑软件还搞了开机时间比分，真是笑话，很多时候电脑开机时间变慢，这些所谓优化软件「功不可没」。

为什么建议你平时不用关机？

1、因为我想要像用 iPhone 和 iPad 那样用 Mac，即用即开。只要打开盖子或按一下电源键就能马上使用了。秒开，瞬间感觉绳命又比别人延长了 60 秒！

2、延续性好。能立即回到之前的工作状态。理想状态下，第一天离开电脑时的桌面布局和软件开启状态，甚至光标聚焦位置都应该保持不变，这也是睡眠的一个优势。

3、爽。试想突然有急事要离开，直接盒上笔记本的盖子，立马出发是多好的体验。同样，有急事需要打开电脑的时候，立马开盖就可以了使用了。

如何让Mac睡眠？

1.如果用的是 Mac 笔记本， 合上盖子就可以了！

2.或点击左上角 ，点选「睡眠」。

3.Mac 台式机按住电源按钮 1.5 秒（一直按会让 Mac 强制关机）

不同 Mac 的电源位置不太一样。笔记本一般是最右上角的按键，台式机一般在机身背面。

总结：推荐用睡眠来代替关机，反正不要把时间浪费在开关机上就对了。

2、Mac需要杀毒软件吗

Mac 上的病毒确实比 Windows 少很多。这得益于 macOS 天生（基于 Unix）的稳定性和安全性比较高，并且 Mac 软件采用沙盒机制，正常安装方式和来源的软件一般都比较安全。

另一方面，也是因为 Mac 市场占有率远低于 Windows PC，黑客针对 Mac 编写病毒、木马、恶意软件的动机不足。

所以正常使用和安装软件不需要杀毒。

让 macOS 更安全的几个小建议：

及时更新系统，macOS 的几乎每次更新都会有安全性的提升以及安全漏洞的修复。使得 macOS 成为越来越安全的操作系统。

打开网络防火墙。在「系统偏好设置」-「安全与隐私」中开启。

不要轻易连接公共 Wi-Fi。

另外，只从 App Store 和信任靠谱的软件官网下载软件，不从第三方平台下载安装所谓破解、激活软件。这点非常重要。很多恶意软件都是在你安装了诸如 Adobe 激活软件进而安装到你的系统里。

3、让你的 Mac 更安全

安全分为数据安全和隐私安全。

数据安全就是如何让你的 Mac 里的文件资料不会轻易丢失。建议开启 iCloud 同步。重要且不是特别占存储空间的文件可以放在 iCloud Drive 里。iCloud 可以备份和在所有苹果设备之间同步你的照片、通讯录、日历、Safari 数据、备忘录甚至桌面上的文件等。同时建议开启 iCloud 双重认证。

同时可以用「时间机器」来整体备份你的 Mac。「时间机器」通过外置存储（如移动硬盘，NAS）对你的 Mac 的所有文件和整个状态进行自动定期备份。一旦系统遇到问题，可以通过之前的「时间机器」备份回到备份时的那个时刻。是不是很神奇。

隐私安全是个大话题，这里只讲方法。

首先要给你的 Mac 设个安全的密码，建议是 6 位，也不要太容易被猜到，重要的是不要忘记这个密码，不然会很麻烦。带触控 ID 的 MacBook Pro 还可以设置指纹密码解锁。还建议设置 Mac 睡眠或开屏保时一段时间后自动锁定 Mac。

开启「文件保险箱」。我们电脑的硬盘对外相当于是敞开大门的，即使关机或者锁定账户的情况，磁盘里的资料仍然是可以读取的。开启「文件保险箱」相当于给你的 Mac 整个磁盘上一道锁，即是电脑丢失，里面的全部内容资料也是安全的。

开启时会要求设置登录密码，并且会生成恢复密钥。请一定不要忘记 Mac 登录密码，并保存好恢复密钥。如果两个同时丢失或忘记，那 Mac 里的所有文件就找不回来了。所以使用这个功能一定要谨慎。

建议开启「防火墙」。如果说「文件保险箱」是保护你的磁盘安全的一道锁，「防火墙」则是保护你的网络安全的一道墙。它允许你阻止某些传入连接，减少 Mac 遭受恶意程序威胁的可能。

开启「查找我的 Mac」。类似于 iOS 上的「查找」，在 Mac 上也可以开启「查找我的 Mac」。开启后就可以通过 iCloud 网页或者 iOS 设备上的「查找」app，远程找到 Mac 的定位位置。在 macOS Catalina 中，即使 Mac 没有连上网络，也可以查到 Mac 的位置。嗨可以给丢失的 Mac 上发送显示一条信息，也可以远程清除掉丢失 Mac 上的数据。

查看哪些软件和服务访问了你的位置信息，并关闭不希望获取自己位置的软件或服务。你也可以选择关闭 Mac 的定位功能。

对于照片、摄像头等敏感的资料和权限，也可以查看和设置。

4、清洁保养

一台 Mac 毕竟不便宜，注意日常使用时候的清洁保养，不仅更美观，也能用的更久。

日常清洁 Mac 的正确姿势

Mac 使用一段时间，屏幕往往容易留下指纹、灰尘，键盘也有油腻腻的，又脏又难看。出于卫生和美观，还是要过一段时间就清洁一下。

清洁屏幕前建议先关机并拔掉电源，再拿无绒软布用水蘸湿擦屏幕。

有几点需要注意的：

•不要使用不明的喷雾剂、清洁剂等。如果不确定，就只用水浸潮或浸湿擦布，不用任何清洁剂，也不要直接把水直接喷在屏幕上。。•不要用嘴在屏幕上哈气 🙄。•水最好用蒸馏水（比如某臣氏蒸馏水），纯净水也行，最好别用自来水。•尽量不要用家用抹布或普通纸巾清理，否则可能会刮花或有绒毛。

清洁机身时也是建议先关机并拔掉电源，使用湿润的无绒软布擦拭电脑外壳。注意不要弄湿或弄潮接口。擦布我自己用 3M 思高，便宜好用。

顺便唠叨一点，平常用电脑时，旁边不要放盛水的水杯或者将水杯和电脑保持安全距离，不然就用盖好盖子的水杯。一定要注意进水不在保修范围。

清洁键盘时可以用吹风机吹掉键盘之间和键盘与显示屏连接处缝隙的灰尘细碎，也可以用旧的软毛牙刷清扫。再用清洁机身同样的方式擦拭就好了。

有看到说用键盘清洁泥或者蓝丁胶来粘掉表面灰尘或者键盘之间残渣，也可以试试，但比较费，我一般不用。

如何正确的给 Mac 笔记本充电？

用苹果官方的建议就是：你可随时随需为 Apple 锂离子电池充电。这个「随时随需」很好，想充就充，需要充的时候就充。我的建议是「佛系充」。

•新 Mac 笔记本第一次建议充满。•充电周期不是充电次数，而是放电量达到电池容量的 100%，才算完成了一个充电周期。总之随时随需，想充就充。总之电池这个东西，不怕你用，就怕你不用。•电量太低时要充。•长时间处于温度太高或太低的环境会影响电池寿命。•长期不用记得充些电再收起来，每过一个月记得拿出来充充电，用一用。•不要一直插着电源用，也不要每次都把电池用完再插上电用。•如果是繁重工作需要大量消耗电量，应该插电源。•锂电池没那么娇气，佛系充。

不必把充电想得那么复杂。笔记本是拿来用的，也不必过于娇气。

夏天 Mac 太烫怎么办？

开空调啊，我不是开玩笑。

然后打开「活动监视器」看看谁在过高的占用 CPU，如果这个软件暂时不需要运行，就关掉。

MacBook Pro 有两个风扇，通风口位于两侧和背面。MacBook Air 电脑的通风口位于背面转轴附近。要保持这些通风口附近不要有东西遮挡住。不要把 Mac 笔记本放在太软的东西上用，比如沙发、枕头、床或膝盖上，会妨碍散热。

还可以买个散热支架，淘宝很多，不用太贵的就行。

另外，雷雳 3（USB-C）接口的 Mac 笔记本扩展坞/转换器会引起不必要的发热并且也会挡住进风口。我的方案是常用到接口时尽可能用线代替扩展坞。必须要用到扩展坞时，用完就拔掉。

比如我的 2018 款 MacBook Pro 连接一台 4K 显示器就用的 USB-C 转 HDMI 线，而不是用扩展坞再连接显示器线。

浏览器的 Flash 播放器往往也是发热的元凶。Flash 效能极低，会耗费大量的系统资源，导致电脑温度快速上升，多出现在网站的视频播放、广告弹窗中。如果安装了 Flash，建议卸载。

好在现在很多视频网站都采用 HTML 5 视频播放模式，或者同时提供 HTML 5 和 Flash 两种模式。尽量选择用 H5 模式播放。省电又不烫。
